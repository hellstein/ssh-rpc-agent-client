# How to create tasks ?

```json
[
    {
        "Topic": "interaction test",
        "Tasks": [
            "vim anything.test",
            "top"
        ]
    },
    {
        "Topic": "sudo test",
        "Tasks": [
            "sudo iftop"
        ]
    },
    {
        "Topic": "random job",
        "Tasks": [
            "cd where",
            "./doanything.sh parameters"
        ]
    }
]
```
